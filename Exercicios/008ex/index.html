Principais formataçoes

Semantica = Tem Sentido.

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag B (nao semantica).

Nesta frase, temos um termo em destaque usando a tag STRONG (semantica).

Itálico / Ênfase

Nesta frase, temos um termo em italico usando a tag I (nao semantica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto marcado

Podemos criar tambem um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

A tag BIG está obsoleta (recomendado alterar no CSS) enquanto a tag SMALL pode continuar sendo usada.

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Texto sobrescrito

Para inserir coisas do tipo x20+3, usar a tag SUP

Texto subscrito

Para inserir coisas do tipo H20, usar a tag SUB
